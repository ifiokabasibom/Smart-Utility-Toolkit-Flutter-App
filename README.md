# Smart Utility Toolkit (Flutter)

A multi-purpose utility mobile application built with Flutter.

## Features

### Unit Converter
Supports:
- Length conversion
- Weight conversion
- Temperature conversion

### Calculator
Basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division

### Notes Module
- Add notes
- Delete notes
- Persistent storage using SharedPreferences

## Architecture

Feature-based folder structure:

lib/
 ├── main.dart
 ├── routes.dart
 └── features/
     ├── converter/
     ├── calculator/
     └── notes/

## Technologies Used

- Flutter
- Dart
- SharedPreferences
